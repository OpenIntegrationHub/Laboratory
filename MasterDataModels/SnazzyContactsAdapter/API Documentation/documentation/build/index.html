POST
Create person

To create a person you have to pass at least first name and last name.

POST mp_contact/json_respond/address_contactperson/json_insert

Optional parameters

Name /type

Description /example

Constraints

firstname
string

First name

"John"

    name
    string

    Last name

    "Doe"

      cURL

      curl -X POST "https://snazzycontacts.com/mp_contact/json_respond/address_contactperson/json_insert" \
     -H "Content-Type: application/json" \
     -H "X-API-KEY: [YOUR_API_KEY]" \
     -H "mp_cookie: dqerpzf8bpj7yoc9kryzbf8zx8e4funj" \
     --data '{"firstname":"John","name":"Doe"}'

      Response

      {
  "rowid": "197775",
  "firstname": "John",
  "name": "Doe"
}

      POST
      Update person

      To update a person you need to pass person's rowid and the field whih you want to update.

      POST mp_contact/json_respond/address_contactperson/json_update

      Optional parameters

      Name /type

      Description /example

      Constraints

      rowid
      string

      User's rowid and field to update

      "197775"

        name
        string

        User's name to update

        "Smith"

          cURL

          curl -X POST "https://snazzycontacts.com/mp_contact/json_respond/address_contactperson/json_update" \
     -H "Content-Type: application/json" \
     -H "X-API-KEY: [YOUR_API_KEY]" \
     -H "mp_cookie: dqerpzf8bpj7yoc9kryzbf8zx8e4funj" \
     --data '{"rowid":"197775","name":"Smith"}'

          Response

          {
  "rowid": "197775",
  "firstname": "John",
  "name": "Smith"
}

          POST
          Delete person

          To delete a person you have to pass person's rowid, which you want to delete. We do NOT really delete the person from our database, we just update the value in field is_deleted to 1 which actually hides the person from the view.

          POST mp_contact/json_respond/address_contactperson/json_update

          Optional parameters

          Name /type

          Description /example

          Constraints

          rowid
          string

          User's rowid which must be deleted

          "197775"

            cURL

            curl -X POST "https://snazzycontacts.com/mp_contact/json_respond/address_contactperson/json_update" \
     -H "Content-Type: application/json" \
     -H "X-API-KEY: [YOUR_API_KEY]" \
     -H "mp_cookie: dqerpzf8bpj7yoc9kryzbf8zx8e4funj" \
     --data '{"rowid":"197775"}'

            Response

            {
  "rowid": "197775",
  "firstname": "John",
  "name": "Doe"
}
